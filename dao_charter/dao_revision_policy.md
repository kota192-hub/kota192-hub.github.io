# DAO憲章 見直し方針

2026年版への改定に向けた基本方針と、コメント採用基準をまとめます。

---

## 1. 基本方針：憲章とガイドラインの住み分け

今回の最大の論点は「何を憲章に書き、何をガイドラインに書くか」です。

### 憲章に書くこと（Why・What）

| カテゴリ | 内容例 |
|---------|--------|
| MVV | ミッション・ビジョン・バリュー |
| 目的・テーマ | 「長崎のうまいサカナの未来をつくる」 |
| 行動指針 | 5つの指針 |
| 基本定義 | メンバー・ホルダーの定義 |
| 権利・原則 | 財産権・意思決定権の原則 |
| 意思決定の範囲 | 何を投票で決めるかの列挙 |

### ガイドラインに書くこと（How）

| カテゴリ | 内容例 |
|---------|--------|
| 参加手続き | メンバー登録・ホルダー取得フロー |
| 投票プロセス詳細 | 起案〜告知〜実施の具体手順 |
| ツール指定 | 使用するDiscordチャンネル・投票ツール |
| ロール権限の詳細 | 分科会の立ち上げ手順など |
| 退会・除名手続き | 具体的な手順 |

## 2. コメント採用の基準

コメントを採用するかどうかは、以下の基準で判断します。

### 採用優先度：高

- **Tycheさんが👍したコメント** → 管理者承認済みと判断
- **くまさん提案（[o][p]）** → 実態に合った例外規定。賛同([q])あり
- **文言修正提案で改定文言が明示されているもの** → 修正内容が明確

### 採用優先度：中（要確認）

- **「解決済み→再び開いた」もの（[n][y]）** → 何を追加するか内容確認が必要
- **[k]「こちらを追加する」** → 具体的な追加内容を確認する必要あり
- **[s][t]リワード制定への投票要否** → 意見が割れているため要議論

### 採用優先度：低（様子見）

- **[u]「NFT投票違反」の条文** → 意味が不明瞭。削除か大幅修正を検討

## 3. 項目別の変更方向性

| 項目 | 変更方向 | 主な変更内容 |
|------|---------|------------|
| **1. 基本的事項** | 大幅削減 | ルール部分をGLへ移動。財産権・移行期間のみ憲章に残す |
| **2. 組織目的と目的** | 格上げ | MVV・行動指針・基本目的を憲章第1章に移動 |
| **3. コミュニティ構成** | 分割 | ロール定義は憲章に残す。参加手続き詳細はGLへ |
| **4. ロールの権限** | 文言修正 | ホルダー権限の文言を明確化。詳細はGLへ |
| **5. 意思決定方法** | 例外追加 | 名称使用・単発リワードの例外規定を追加 |
| **6. 投票ルール** | 簡略化 | 定足数に「休眠ホルダー除く」を明記。手続き詳細はGLへ |
| **7. 投票ルール（ルール変更）** | 文言修正 | 「特別決議」として再定義 |
| **8. トークン** | 現状維持 | Season3移行方針が決まるまで現状維持 |
| **附則** | 現状維持 | 施行日のみ更新 |
